How to prompt

Write role notes in plain text — the server wraps them in the model's own chat template, so you never type turn markers or BOS/EOS. Lines starting ### SYSTEM:, ### USER: or ### ASSISTANT: are notes; every other line is your prose. Generate continues from the cursor.

### SYSTEM: You are a co-writer. Match the voice and style.
### USER: Write a story about a monkey. Ornate prose.
Once upon a time, beneath the gilded canopy…
### USER: Write tersely now. Short sentences.
The monkey woke. He▮
  • Wrapping a note over several lines? Repeat the tag on each line.
  • Output stops when the model ends its turn — often a sentence or two. Hit Generate again to keep going.

The document is sent verbatim — type the model's chat-template turn markers yourself, then Generate with the cursor after the open assistant turn (ChatML shown; Llama-3 uses <|start_header_id|>…<|eot_id|>).

<|im_start|>system
You are a helpful assistant.<|im_end|>
<|im_start|>user
Write a haiku about the sea.<|im_end|>
<|im_start|>assistant

Each Generate adds up to 512 tokens (set by the writer model's Max Tokens).
